# 👋 Hi there! 👋

My name is Marco, I am from Poland and I'm an aspiring software engineer!

## ✨ Current project ✨

The current project where I spend the most time on is sadly private, so it's not visible on GitHub :(

It's a Java/Spring Boot/React stack of applications, where I basically made a overlay over a website's API to make my own website, because they did not have the tools I wanted them to have. So I made my own tools for this. It comes with a full frontend written in React, and is portable using a Docker Compose setup. The backend handles the requests, aggregates the data and compares it to previous data/reference data, and the frontend is basically a very cozy GUI for me to use this on.

One of my favorite public projects is the recipe-application project. It's a mobile application based on **Flutter**, which allows you to browse and search for recipes, add them to your favorites, manage your fridge content, and allows for searching the database for recipes including chosen ingredients. It's my favorite, because I've teached myself Flutter for this project and overall it was a blast to work on it.

The recipes on the application are on two different locations: my own **Firebase NoSQL database**, and pulled from the **Spoonacular API**. They are pulled from both locations, and unified in one DTO in a **Spring Boot API** to allow for easy display on the application.

My main focus currently is to solidify my Java/Spring Boot knowledge and expand it with adjacent knowledge domains, such as Docker, Kafka and cloud.
